# ⚽ Premier League Analytics Dashboard

## 📌 Project Overview
An end-to-end data analytics project analyzing 31 seasons (1993–2024) 
of Premier League data using SQL, Python, Power BI and Excel.

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| PostgreSQL | Database design & SQL analysis |
| Python | Automated ETL data pipeline |
| Power BI | Interactive 5-page dashboard |
| Excel | Data cleaning & pivot analysis |

## 📊 Dashboard Pages
1. **League Overview** — Standings, KPI cards, season slicer
2. **Team Performance** — Goals, wins, points comparison
3. **Player Stats** — Top scorers, assists, nationality breakdown
4. **Match Analysis** — Results, home vs away, trends
5. **Season Trends** — Title winners, points history

## 🗄️ Database Schema
- 7 tables: seasons, teams, standings, matches, club_stats, players, player_stats
- 12,026+ match records
- 1,116 player records
- 646 season standings rows

## 📝 Key SQL Concepts Used
- Window Functions (RANK, ROW_NUMBER, LAG)
- CTEs (Common Table Expressions)
- Multiple JOINs across 7 tables
- CASE statements
- Aggregate functions
- Stored Procedures

## 🔍 Key Insights
- Manchester United won the most Premier League titles (13)
- Mohamed Salah was top scorer in 2024/25 with 29 goals
- Home teams win 45.89% of all matches since 1993
- England has the most players in the Premier League
